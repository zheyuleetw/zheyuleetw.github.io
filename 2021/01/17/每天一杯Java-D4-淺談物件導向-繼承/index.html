extends關鍵字的使用

首先，Java中使用extends關鍵字有幾個規則：

  1. 一個類可以被多個類繼承
  2. 一個類只能繼承一個父類，即所謂單繼承
  3. 可以分為間接繼承與直接繼承，假設有三個類ABC，B繼承於A，C繼承於B，則AB的關係是直接繼承，AC的關係則為間接繼承。
